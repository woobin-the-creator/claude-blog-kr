이제 Microsoft Foundry에서 Claude를 정식으로 사용할 수 있습니다

2026년 6월 29일 · 카테고리: Enterprise AI · 읽는 데 5분 원문: Claude in Microsoft Foundry is now generally available (한글 번역본)

오늘부터 Claude 모델을 Azure에서 호스팅되는 Microsoft Foundry에서 정식으로(generally available) 사용할 수 있습니다. Claude는 여러분의 Azure 환경에서 실행되며, 여러분의 팀이 이미 사용하고 있는 인증·과금·거버넌스 제어를 그대로 활용합니다. 데이터 레지던시(data residency) 요건이 있는 팀을 위한 미국 데이터 존(US data zone)을 포함해, 추론(inference)이 처리되는 위치를 선택할 수 있습니다. 추론은 Anthropic이 운영하며, Anthropic이 데이터 처리자(data processor)입니다.

여러분의 Azure 계정으로 Claude를 활용해 빌드하기

시작 시점에는 Claude Opus 4.8과 Claude Haiku 4.5를 Messages API에서 사용할 수 있으며, 프롬프트 캐싱(prompt caching)과 확장 사고(extended thinking)와 같은 핵심 기능을 함께 제공해 코딩과 에이전트 작업부터 복잡한 추론까지 다양한 활용 사례를 지원합니다. 앞으로도 Foundry에서 사용할 수 있는 범위를 계속 확대해 나갈 예정입니다.

Microsoft Foundry의 Claude는 Azure 네이티브로, 여러분이 이미 쓰고 있는 Azure 아이덴티티·네트워킹·거버넌스 제어와 함께 동작합니다. 통합된 단일 청구서를 받게 되며, Microsoft Enterprise Agreement를 보유한 적격 고객의 경우 Claude 사용량이 Microsoft Azure 약정(commitment)에서 차감됩니다.

Anthropic이 운영하는, Azure 위의 Claude 실행

Microsoft Foundry에서 Claude를 실행하는 방법은 두 가지입니다. 여러분의 Azure 환경에서 실행하는 것이 중요하다면 Azure에서 호스팅(hosted on Azure)을 선택하세요 — Azure 인증·과금·거버넌스와 미국 데이터 존이 함께 제공됩니다. 전체 API 기능이 필요하거나 아직 Azure에서 제공되지 않는 모델이 필요하다면 Anthropic에서 호스팅(hosted on Anthropic)(이전의 Foundry Preview)을 선택하세요. 시간이 지나면서, 우리는 Azure 호스팅 제공 방식과 Anthropic 호스팅 방식 사이에 기능과 모델의 동등성(parity)을 확보하는 것을 목표로 하고 있습니다.
